# MP DOORS One-Page Website

A clean one-page B2B presentation website for MP DOORS focused on door installation services (B2B), project references, partners, and contact.

## Features
- Sticky navigation with mobile drawer menu
- Hero section with main project photo
- Services grid
- References / projects photo grid with lightbox
- Partners section with logos and short descriptions
- Process section (how cooperation works)
- Contact section with form layout
- Scroll reveal animations (via `reveal.js`) and others..

## Tech Stack
- HTML
- CSS (split into multiple files)
- Vanilla JavaScript

- Live on: https://mp-doors.netlify.app/
